# EmvData

This class holds information about the EMV data necessary for transaction request.

### PROPERTIES

`cardExpiry`: String,

`cardPIN`: String,

`cardPAN`: String,

`cardTrack2`: String,

`src`: String, // service restriction code

`csn`: String, // card sequence number

`AID`: String,

`icc`: IccData,

`pinKsn`: String
